    title: Hexo Instruction
    date: 2020-11-12 21:56:44
    tags: Instructions

    This article only deals with how to handle daily operation like creating a new blog, managing the homepage, and uploading the static page onto Github. More information click installation or operation.

    Open the local website

    Open Terminal, and type in:

    $ cd desktop

    Depending on where you keep your files, it could be [desktop], [download], [document] or the like.

    $ cd MyFirstHexoBlog

    These two steps are to locate the file where your website is saved. [MyFirstHexoBlog] is the name I give to the folder where I keep my website file. Be sure to change it to yours when following the instructions.

    $ hexo server

    So now you can copy the link generated by Terminal and paste it on a Browser to see your website.

    Create a new Blog

    $ hexo new "My New Post"

    Once this step is done, a new file is automatically created into the original folder, and now your can write your blog with the help of Visual Studio Code. Go to [source/posts] and it's the item that ends with [.md]. Remember to keep one original file for reference.

    Install a theme

    Go to Hexo official website and look for a theme you like. Once you find a proper one, go to its Github page and copy its URL. Open Terminal and type in:

    $ git clone https://**** themes/themename

    Then it's downloading the theme to your website folder, and saving it under [theme] by the name of [themename] or other name you prefer. Once it's downloaded, go to Visial Studio Code, open [config.yml] under the themes folder, locate the line related with theme name and change it to the new one. Restart the server once finished.
